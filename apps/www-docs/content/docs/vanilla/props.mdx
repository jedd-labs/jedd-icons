The vanilla package exposes the shared styling model as `createElement`
options, plus a couple specific to imperative rendering and the `createIcons`
helper.

## createElement options

  type={{

## Variant selection

In vanilla you select the variant with the `variant` option on `createElement`
(or via `attrs.variant` in `createIcons`) — `"stroke"` (default) or `"fill"`.
Import filled icon data from the `@jedd-icons/core/fill` subpath.
